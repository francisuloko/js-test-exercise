# js-test-exercise

> An exercise to test JavaScript functions using the Jest library

## How to run test

- `git clone https://github.com/francisuloko/js-test-excercise.git`
- `cd js-test-exercise`
- `npm init`
- `npm install --save-dev jest`
- `add { scripts : { "test: "jest" } } to package.json`
- `npm run test`


## Prerequisite
- npm
- yarn
- terminal
